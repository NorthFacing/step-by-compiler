
* 一棵语法分析树是具有如下性质的树
    - 根节点的标号为文法的开始符号，有且仅有一个根节点
    - 没有子节点的节点称为`叶子节点`，每一个叶子节点的标号为一个终结符号或 `ε`
    - 有一个或多个子节点的节点称为`内部节点`，每个内部节点的标号为一个非终结符号
* 每个内部节点和子节点都对应一个产生式
    - 内部节点对应产生式的头
    - 它的子节点对应产生式的体
