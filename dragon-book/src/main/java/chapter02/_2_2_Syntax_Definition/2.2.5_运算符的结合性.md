
* 首先注意：结合性规则只能解决同一个运算符多次出现的情况，出现不同运算符的运算顺序还需要运算符的优先级规则来确定
* 左结合
    - 当一个运算分量左右两侧都有"+"号时，它属于其左边的运算符
    - 加、减、乘、除 都属于左结合
* 右结合
    - 指数运算
    - 赋值运算及其后裔（=、+=、-=）
* 将文法规范进行进化，添加结合性规则进去
