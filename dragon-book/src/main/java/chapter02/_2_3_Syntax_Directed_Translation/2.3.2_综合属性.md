
* 注释语法分析树
    如果一棵语法分析树的各个节点上标记了相应的属性值，那么这棵语法分析树就被称为注释语法分析树，简称注释分析树。
* 综合属性
    - 如果某个属性在语法分析树节点N上的值是由N的子节点以及N本身的属性值决定的，那么这个属性就被称为综合属性。
    - 综合属性有个很好的性质：只需要对语法分析树进行一次自底向上遍历，就可以计算出属性的值。
* 一个注释语法分析树和综合属性范例
    ```
    9-5+2
    ```
