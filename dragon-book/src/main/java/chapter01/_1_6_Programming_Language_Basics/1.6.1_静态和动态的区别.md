
* 静态的动态的去呗
    - 如果一个语言使用的策略支持编译器静态决定某个问题，那么我们说这个语言使用了一个静态策略，
或者说这个问题可以再编译时刻决定。
    - 一个只允许在运行程序的时候做出决定的策略被称为动态策略，或者被认为需要在运行时刻做出决定
* 作用域
    - 静态作用域：仅阅读程序就可以确定一个声明的作用域（C，Java等大部分语言）
    - 动态作用域：阅读时不能确定作用域
* Java中的static关键字
    - 如：public static int x;
    - 使得变量x成为类变量，不管多少个实例都共享这一个变量
    - 编译器可以确定内存中存放此变量的位置
